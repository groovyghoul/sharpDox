Start

ToSvg

Deklariert von: SharpDox.UML.ISDDiagram

Syntax

public abstract String ToSvg(Double maxWidth)

Beschreibung

Diese Methode erstellt eine SVG-Repräsentation des Diagramms.

Rückgabe

Die SVG-Repräsentation

Parameter

maxWidth

Falls das Diagramm breiter als der Wert ist, wird das Diagramm auf diesen Wert skaliert.
