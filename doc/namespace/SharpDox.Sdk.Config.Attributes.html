Name Description
ExcludeAttribute

Konfigurationselemente die mit diesem Attribut gekennzeichnet wurden werden kein Teil der Konfigurationsdatei die von sharpDox gespeichert wird.
RequiredAttribute

Konfigurationselemente die mit diesem Attribut gekennzeichnet wurden werden in der GUI als 'erforderlich' markiert.
ConfigEditorAttribute
NameAttribute

Falls ein Konfigurationselement in der GUI sichtbar sein soll, muss es mit diesem Attribut gekennzeichnet sein. Damit bekommt es einen Anzeigenamen für das Propertygrid.
EditorType

Alle möglichen Editortypen für die GUI (außer den standard Controls für strings & booleans).
