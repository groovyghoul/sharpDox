public interface IConfigController

Beschreibung

Dieses Interface bildet alle Funktionen ab, die zur Arbeit mit Konfigurationsdateien notwendig sind.

Ereignisse

OnRecentProjectsChanged

Feuert, falls ein neues Projekt zu der List der zuletzt benutzten Projekte hinzugefügt wurde.

public OnRecentProjectsChanged

Methoden

GetAllConfigSections()

Holz alle Konfigurationssektionen der aktuellenKonfigurationsdatei.

public abstract IEnumerable<IConfigSection> GetAllConfigSections()

Rückgabe

Gibt alle Konfigurationssektionen zurück.

GetConfigSection()

Holt eine spezifische Konfigurationssektion der aktuellen Konfigurationsdatei.

public abstract T GetConfigSection<T>()

Beispiel

var config = _configController.GetConfigSection<SharpDoxConfig>();

Rückgabe

Liefert die Konfigurationssektion zurück, falls diese existiert. Ansonsten null.

T

Eine List der zuletzt benutzten Konfigurationsdateien. Limitiert auf die letzten fünf. Der Key des KeyValuePairs ist der Pfad und der Value ider der Name der Konfigurationsdatei.
