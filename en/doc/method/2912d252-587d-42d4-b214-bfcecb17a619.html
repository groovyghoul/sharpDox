GetSequenceDiagram

Declaringtype: SharpDox.UML.DiagramExtensions

Syntax

public ISDDiagram GetSequenceDiagram(SDMethod method, List repositoryTypes)

Summary

Creates the sequence diagram for the given SDMethod.

Returns

The sequence diagram for the given SDMethod.

Parameters

method

Create the sequence diagram for this SDMethod.

repositoryTypes

The method needs all known types of the current solution. You are able to get it with the GetAllTypes method of the SDRepository.
