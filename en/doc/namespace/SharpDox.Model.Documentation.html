Description

The SharpDox.Model.Documentation namespace contains all types for parsing the inline documentation. SDDocumentation represents an inline documentation with all it members. For example the summary or example tag. Every property of SDDocumentation is a list of documentation tokens. For more information please see SharpDox.Model.Documentation.Token.
